Hyperscript

Hyperscript is a new client side language which adds functionality to web applications.

Based on the popular HyperTalk language developed for Apple's HyperCard application, HyperScript combines an English-prose syntax with powerful event management.

Advantages of using HyperScript in the browser include

  • Self documenting syntax with prose style readability
  • Event listening built in by using simple 'on eventname' construct
  • Custom events can be sent to, and listened for, by any DOM element
  • Asynchronous calls and Promise resolution handled behind the scenes for you
  • Null safety automatically and silently handled

Hyperscript can be used via CDN link in the document head tag
