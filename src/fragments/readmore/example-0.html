Hypermedia is an extension of the term hypertext which is a nonlinear medium of information that includes graphics, audio, and video linked together by hyperlinks. The term was first used by Ted Nelson in an article published in 1965.

The World Wide Web and Bill Atkinson's HyperCard are popular examples of hypermedia.

Hypermedia exchanges between server and client remove the need for a fixed API protocol since the client recieves display-ready html containing hyperlinks which direct the user to other data sources

Hypermedia encourages the server to control application state thus avoiding discord in state between the server and client

HATEOAS (Hypermedia as the Engine of Application State) is a constaint of the REST application architecture whereby a client interacts with the server entirely through hypermedia exchanges.

In theory this means that no API documentation is needed because the client needs no prior knowledge of how to interact with any particular application or server beyond a generic ability to display hypermedia.
