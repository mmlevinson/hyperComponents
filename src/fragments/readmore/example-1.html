HTMX is a small JavaScript library which extends the actions available to html elements. In particular htmx provides a suite of attributes which can be added to any html to enable server requests and target responses trigger by any event.

HTMX extends the core concept of hypermedia exchanges between client and server by allowing

  • Any element can issues HTTP requests
  • Any event can trigger an HTTP request
  • Any HTTP verb (not just GET or POST) can be used
  • Any event may trigger an HTTP request
  • Any HTTP response may be targeted to any element

To utilize HTMX on the client side, all that is needed is script tag to the CDN source code placed in the document's head

Any HTML element can issue HTTP requests to the server by including any of the following attributes

  • hx-get
  • hx-post
  • hx-put
  • hx-patch
  • hx-delete

The server must responde with hypermedia instead of JSON. Fully constructed HTML fragment sent by the server (plus HTMX specific headers) allows the browser to selectivley update any part of the DOM using the response hypermedia.
