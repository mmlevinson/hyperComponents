HTMX is a small JavaScript library which extends the actions available to html elements. In particular htmx provides a suite of attributes which can be added to any html element in order to enable server requests and target responses triggered by any event.

HTMX extends the core concept of hypermedia exchanges between client and server by allowing

  • Any element to issue HTTP requests
  • Any event can trigger an HTTP request
  • Any HTTP verb (not just GET or POST) can be used
  • Any event may trigger an HTTP request
  • Any HTTP response may be targeted to any element

To utilize HTMX on the client side, all that is needed is script tag to the CDN source code.

Any HTML element can issue HTTP requests to the server by including any of the following attributes

  • hx-get
  • hx-post
  • hx-put
  • hx-patch
  • hx-delete

The server must respond with hypermedia instead of JSON. Fully constructed HTML fragments sent by the server (plus HTMX specific headers) allow the browser to selectively update any part of the DOM using the response hypermedia.
