### Description

This example demonstrates a commonly used web component that presents simple button which displays a list of choices when activated by a click or tap.

Common usage includes display of 

The choice list appears as a dropdown below (or in a defined location near) the button.   

A Click/Tap on a menu item triggers a 
a custom event to be fired followed by dismissal of the dropdown.

### Features

The trigger button and the dropdown are styled with Tailwind CSS utility classes and are both
fully customizable using props (see below)

The action triggered by tapping a menu item is customizable.

Appearance and disappearance of the dropdown are both smoothly animated using HyperScript

### Props

```ts
interface Props {
 id:string 
 caption:string   
 dropdownStyles:string   
 buttonStyles:string   
 listStyles:string   
 moveXY:string 
 duration: string
}

const {id, caption, dropdownStyles, buttonStyles, listStyles, moveXY = "0,0", duration = '100ms'} = Astro.props
```
